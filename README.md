# Sahyadri-Samrakshane

Sahyadri-Samrakshane is a **Forest Sentinel** Android application for reporting ecological alerts in the Western Ghats. It helps trekkers, villagers, and local communities report forest fires, landslides, illegal tree cutting, and wildlife sightings with photo proof and GPS coordinates.

## Problem Statement

The Western Ghats are eco-sensitive zones. Local communities often notice environmental threats early, but they may not have a direct way to report incidents to the forest department with exact coordinates. Delayed reporting can lead to larger damage.

## Solution

Sahyadri-Samrakshane allows users to submit ecological alerts with:

- Alert type
- Captured photo
- High-accuracy GPS coordinates
- Short note
- Offline caching
- Sync-ready dashboard status

## Features

- Secure login
- Wrong-login owner alert
- CameraX photo capture
- FusedLocationProviderClient GPS location
- GPS coordinates displayed during reporting
- Alert types:
  - Forest Fire
  - Landslide
  - Illegal Tree Cutting
  - Wildlife Sighting
- Alert statuses:
  - Reported
  - Verified
  - Team Dispatched
- Offline queue for low-signal areas
- Mock Firebase dashboard
- Eco-sensitive zone education tips
- Nature-friendly green and earth-tone UI
- Browser-based live demo page
